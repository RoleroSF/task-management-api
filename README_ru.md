# Task Management API

RESTful API для управления задачами с аутентификацией и авторизацией.

## Стек технологий

* ASP.NET Core Web API (.NET 10)
* Entity Framework Core
* PostgreSQL
* JWT Authentication
* Serilog (логирование)
* Swagger (документация API)

## Функциональность

### Аутентификация

* Регистрация пользователя
* Логин с выдачей JWT токена
* Хеширование паролей

### Авторизация

* Роли: `user`, `admin`
* Ограничение доступа через `[Authorize]`
* Разграничение данных по пользователю

### Управление задачами

* CRUD операции
* Привязка задач к пользователю
* Pagination
* Filtering
* Sorting

### Администрирование

* Просмотр всех пользователей (только admin)
* Редактирование пользователей

### Дополнительно

* Глобальный обработчик ошибок
* Логирование через Serilog
* Swagger UI с поддержкой JWT

---

## Архитектура

* Controller → Service → Data (DbContext)
* DTO для разделения API и модели данных
* Mapper для преобразования сущностей
* Middleware для обработки ошибок
* Dependency Injection

---

## Аутентификация в Swagger

1. Выполнить `POST /api/auth/login`
2. Скопировать токен
3. Нажать `Authorize`
4. Вставить:`Bearer <token>`
